Questão - 1

Atividade 1 – O que é uma regra CSS?

Uma regra CSS é a unidade básica de estilização. Ela é composta por:

  • Seletor: Define o elemento alvo
  • Propriedade: Característica que será alterada
  • Valor: O estilo aplicado

Exemplo:

p { color: green; font-size: 18px; }

Atividade 3 – Formas de incluir CSS

  • Inline: Atributo style direto na tag
  • Interno: Tag <style> dentro do <head>
  • Externo: Arquivo .css via <link rel="stylesheet">
