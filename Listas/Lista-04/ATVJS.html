O que é JavaScript

Em meados dos anos 90 a web se popularizou e com o crescimento a necessidade de interatividade entere cliente, o que seria possivel uma linguagem de programação que fosse executada no navefador do usuario, tendo acesso ao documento HTMLda dopagina e seus artefatos de codigos

Surgiram diversar propostas na industria, masa que se popularizou doi a linguagem script, criada na NetScape denomidada de ECMA-267, conhecida como popularmente como Java Script

O codigo JavaScript é executado no navefador dentro de um SandBox, que limita as operações e acesso do codigo executado, fornecendo operacões e variaavels limitando o acesso a rede, disco alem de entrada e saida. Esse conceito é fundamental na web , em que código de terceirospodem são baixados e executados na maquina cliente

Quando o interpretador JS executa um script, ele disponibiliza algumas variaveis goblais, que fornecem acesso ao ambiente. No node por exemplo a variavel process force acesso ao sistema operacional e dispositivos de entrada saida. Já no navegador o Java Script é executando em um sandbox que tem acesso a um objeto global denominado windows, que representa a janela e tem dezenas de propriadades que fornecem acesso ao console, document, history, screen e além de funções.
