HTTP - Protocolo de Comunicação da Web

O que é o HTTP?

O HTTP (HyperText Transfer Protocol) é o protocolo responsável pela comunicação entre o navegador e o servidor, permitindo o envio e o recebimento de páginas e outros recursos da Web.

Como funciona?

  • O cliente é o navegador utilizado pelo usuário.
  • O servidor é o computador que hospeda o site.
  • A requisição é o pedido enviado pelo navegador ao servidor.
  • A resposta é o retorno do servidor com a página ou outra informação solicitada.

Métodos HTTP

  • GET: utilizado para solicitar informações ao servidor.
  • POST: utilizado para enviar informações ao servidor, como dados de formulários.

Exemplo de Requisição HTTP

  • Método: GET
  • URL: /index.html
  • Host: www.exemplo.com

Exemplo de Resposta HTTP

  • Status: 200 OK
  • Content-Type: text/html
  • Conteúdo: Página HTML solicitada.

Códigos de Status HTTP

  • 200 OK: requisição realizada com sucesso.
  • 301 Moved Permanently: recurso movido para outro endereço.
  • 400 Bad Request: requisição inválida.
  • 403 Forbidden: acesso negado ao recurso.
  • 404 Not Found: página não encontrada.
  • 500 Internal Server Error: erro interno no servidor.
